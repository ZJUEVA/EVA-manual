# 厂家支持页面

本页列出了一些常见厂家的下载页面。

未列出的品牌可通过搜索“品牌”+“支持”找到下载页面。

## 关于维修手册

现在，部分国家和地区要求数码产品制造商提供维修手册。我们可以在一些跨国品牌的支持页找到这些手册。

很遗憾，并非所有手册都提供简体中文版本。我们可以将语言切换到英文后下载英文版维修手册。

## 页面列表

### 联想（Lenovo）

驱动下载：https://newsupport.lenovo.com.cn/driveDownloads_index.html

重装系统后建议使用“联想驱动安装工具”进行驱动安装。

我们暂时没有找到联想的手册下载页面。

#### Thinkpad

Thinkpad 产品线中的一部分型号可以在联想驱动下载中找到，若找不到，可以访问另一个链接：https://newthink.lenovo.com.cn/driverdownload.html



### 华硕（Asus，含ROG）

驱动、手册下载：https://www.asus.com.cn/support/download-center/

ROG系列部分机型需在单独链接下载：https://rog.asus.com.cn/support/download-center/

部分机型仅提供繁体中文/英文的手册。

### 惠普（HP）

驱动、手册下载：https://support.hp.com/cn-zh

拉到中间，输入型号/序列号查询。部分机型仅提供英文手册。

### 戴尔（DELL，含外星人 Alienware）

驱动、手册下载：https://www.dell.com/support/home/zh-cn/products
