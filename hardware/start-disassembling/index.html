    拆开和装回笔记本

    在清灰换硅脂、加装/更换内存和硬盘等操作之前，我们需要把电脑拆开。本页是拆开笔记本电脑的一般指南。

    0. 拆开笔记本前的准备

    在拆开笔记本之前，我们建议您进行一些准备。

    准备工具

    建议工具：

    • 拆机片
    • 螺丝刀（对应形状和大小）
    关于螺丝刀

    螺丝刀头有不同规格。一般笔记本后盖外侧螺丝有可能为六角星形或十字。拆开后盖后，笔记本内部螺丝一般均为十字。

    如果您平时螺丝刀使用次数较多，您可以购买多合一螺丝刀套装。套装内一般会包含市面上的主流螺丝类型对应的螺丝刀批头。

    可选工具：

    • 吸盘
    • 塑料撬棒
    • 螺丝盘或磁吸板

    除静电

    笔记本电脑主板上的元件较为精密。如果我们身上存在静电，极易击穿主板上的元件。

    您可以通过以下几种方式除去身上的静电

    • 佩戴防静电手环，并将手环连接至有效接地的位置或大金属体
    • 触摸存在有效接地的电器，如电热水壶金属外壳
    • 使用流动的水洗手

    1. 拧开后盖螺丝

    笔记本的四个面

    在维修过程中，我们将笔记本分A、B、C、D四个面。您可以将笔记本想象成两张纸，从上到下为：

    • A面（顶面）
    • B面（屏幕面）
    • C面（键盘面）
    • D面（底面）

    将笔记本D面朝上，取下所有螺丝即可。

    当后盖外裸露螺丝较少时需要注意，有一部分型号笔记本电脑的螺丝会隐藏在脚垫下方。可以在网上查找拆机视频或对应型号的维修手册，确认螺丝数量。

    2. 拆开后盖

    绝大多数笔记本在拆开后盖全部螺丝后，即可取下后盖。

    若后盖较为光滑，可以使用吸盘吸住后盖的一角，固定机身后尝试将后盖提起。提起一条缝后可使用撬片划开剩余卡扣。

    若后盖较为粗糙或使用吸盘无法吸开，可直接使用撬片，寻找一处缝隙较大的位置，将撬片插入后扭动撬片以撬起卡扣。

    部分机型在拆开后盖时需要特殊注意，如：

    • 联想拯救者部分机型后盖延伸至侧边出风口位置，需从侧边撬起。
    • ROG 等高端游戏本C、D面连接处有可能存在灯条。若灯条与D面连接，需从转轴侧抬起后盖，拔去灯条排线后再抬起整个后盖。

    3. 断开电源排线

    电源排线为电池与主板的连接线。

    在拆开笔记本后盖后，做任何进一步操作之前，我们都建议您断开笔记本的电源线。否则，当您的手或者其他导电物体触碰到主板上时，可能导致某些元件短路，从而烧毁其他元件。

    找到电源排线后，将其断开即可。

    不同笔记本断开电源排线的方式

    不同品牌/型号的电脑有不同的断开电源排线的方式。以下举几个例子。

    • 联想、惠普等：直接拔出电源线。
    • 华为部分机型：先抬起电源线接头的一侧，再斜向上抬起排线。
    • 华硕部分机型：先将金属挡片向一侧推开，再垂直向上拔起排线。

    若电池与主板之间无软质电源线连接，则需将整块电池拆下。在拆除电池过程中请勿用手或导电物体触碰到主板。

    4. 释放主板上的残留弱电

    主板上有电容、电感等储能元件。若残留弱电未被完全释放，某些元件短路后依然有可能击穿主板上别的元件。

    仍旧保持D面向上。若电源键在C面上，将屏幕打开一个角度以按到电源键。

    按住电源键5-10秒以释放主板上的残留弱电。

    5. 装回笔记本

    1. 检查在拆开笔记本后断开的线缆、拆下的螺丝是否已全部装回
    2. 插回电源排线
    3. 盖回后盖
    4. 拧回后盖螺丝
    5. 检查能否正常开机

    如果不能正常开机

    如果不能正常开机，请按以下方法排查。

    1. 插上电源线，重试能否正常开机

      一部分笔记本电脑（如华硕天选系列、ROG系列，机械革命部分型号）存在运输模式，需要插上电源适配器后才能正常开机。

      其中，部分型号必须插上原装电源适配器才能开机，而其他型号仅需插上供电（包括USB-PD供电）即可开机。

    2. 重新开盖，检查电源排线有无插紧

    3. 检查其他排线、内存等有无插紧

    以下为玄学方法（在维修过程中遇到过）

    1. 拔掉电源，直接连接电源适配器，查看能否开机；若能开机，重新插回电源后再查看能否开机。
    2. 静置一段时间后，查看能否开机。
